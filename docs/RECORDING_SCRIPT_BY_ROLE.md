# 아버지 (강민수)

동일 성우가 **연령 연기로 두 시점**을 소화한다.
- **1988년(young)** — `b0_tape`, `b4_radio`. 30대 초반. 무뚝뚝하고 말주변이 없다. 애정은 문장 사이 틈으로만 새어 나온다.
- **현재(old)** — `b5_letter`, `b7_1`. 60대 후반. 느리고 갈라진 목소리. 기억이 흐려지는 사람의 호흡.

경상도 억양(김해)을 옅게 깔되 과장하지 않는다. 울먹이지 않는다 — 담담함이 오히려 아프게 들리도록.

# 소영 (강소영, 37세)

**전 구간 동일 성우 필수** — 감정이 이어져야 한다. 분량이 가장 많다.

⚠️ **말투가 중간에 바뀐다(D7).** `b0_call`부터 `b1_b` 앞부분까지는 존댓말,
`b1_b`의 “…고마워. 이제 말 놓을게.”부터 끝까지 반말로 고정한다. 순서를 바꾸면 안 된다.

전화기 너머의 목소리다. 현장에 없고 아버지 곁에 있다 — 걷는 숨소리나 바깥 소음이 들어가면 안 된다.
감정선: 조심스러움 → 친밀 → 울음을 참음(b2_b·b4_b) → 터짐(b5_f) → 밝아짐(b6·b7).
