## 1. 집(새 환경)에서 시작하는 법

   `b0_tape`·`b5_letter`·`b4_radio`)는 성우 권장, 소영은 TTS(클로바 더빙 등) 가능.

2. **⚠️ 실판매(7/24~26) 전 만능 코드 비활성화** — `src/lib/tracks.ts`의
   `UNIVERSAL_PASS_CODE = '1111'` → `null` 한 줄.
3. **피날레·빙고 상세 검증** — 빙고판 진입까지만 클릭 확인했다. 빙고 셀 20종 완료
   → FINISH → B7_0 → 피날레 저장(PNG)·EP.2 버튼은 미검증.
4. **Storage(사진 업로드)** — Blaze 요금제 결제계정 한도 문제로 보류 중.
   해결 시 `PostComposer.tsx`의 `PHOTO_UPLOAD_ENABLED=true` + `storage.rules` 작성.
5. **b0_call 길이 불일치** — 현재 38초 파일 vs 대본 70초. 정식 70초 녹음이 오면
   교체(별칭 제거하고 `b0_call.mp3`로).
6. **Track 4 라이선스곡** — 라디오 끝 페이드인 곡. 원곡 '소녀' 사용 금지(D5·저작권).
7. 소소한 정리: Firebase Auth의 테스트 계정 `clidiag01@bonghwang.local` 삭제,
   구 라우트(`/story`, `/exploration` 링크 정리), 개발모드 지도 중복 초기화.

## 5. 주의사항 (전에 겪은 함정)

- **dev 서버 켜둔 채 `npm run build` 금지** — `.next/`가 덮여 dev가 깨진다.
  빌드 후엔 `.next` 삭제하고 dev 재시작.
- **오디오/캐시 파일 교체 시 `sw.js`의 `CACHE_NAME` 버전 상향** — 안 올리면 기존
  방문 기기가 옛 파일을 계속 쓴다.
- `.claude/settings.local.json`은 커밋하지 않는다 (로컬 권한 목록).
- 대사는 명세서 §6 원문 고정 — 임의 수정 금지. 금지어: '치매'(D6),
  '미움/미워'(D6b) — 빌드가 자동 차단한다(`npm run check:forbidden`).
- QR 6회 원칙(D4): qr 트리거 번들은 정확히 5개(인트로는 user_tap:PLAY).
  `validateCueGraph()`가 검사한다.
