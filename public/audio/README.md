# 봉황1988 EP.1 — 음성 파일 (녹음 발주용, 명세서 v2 §12)

이 폴더에 음성 파일을 넣으면 앱이 자동 인식해 재생합니다.
**파일이 없으면 해당 번들은 자막 전용 모드로 정상 진행**되므로, 준비된 것부터 넣어도 됩니다.

- 형식: `.m4a`(AAC) 우선, 없으면 같은 이름의 `.mp3`를 찾습니다
- 파일명은 아래 표와 **정확히 일치**해야 합니다 (소문자·언더스코어)
- 대사 원문: `src/lib/cues.ts`의 `subtitleLines` (명세서 §6과 동일 — 임의 수정 금지)
- 번들 안의 호흡(사이·pause)은 **오디오 파일 안에서** 구현합니다 (D13)

## 제작 유의 (§12)

- 소영 전 구간 **동일 성우** — 감정 연속성 필수. 존댓말(b0_call~b1_b 전반) → 반말(b1_b 후반부터) 흐름 주의
- 아버지는 동일 성우의 **연령 연기**: young(1988, b0_tape·b4_radio) / old(현재, b5_letter·b7_1)
- 볼륨 -16 LUFS 내외로 통일 (야외 청취)
- 현재 `intro-soyoung.mp3`(38초)는 `b0_call`이 없을 때의 임시 폴백입니다.
  정식 70초 `b0_call`이 오면 교체하세요.

# 대본 전문 (명세서 §6 원문 — 임의 수정 금지)

`[ ]` 대괄호는 연출 지시로 녹음 대상이 아닙니다. 문장 사이 `…`와 `[사이]`는
호흡으로 살려주세요. 번들 안의 사이(pause)는 오디오 파일 안에서 구현합니다(D13).
